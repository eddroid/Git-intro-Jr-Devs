# Git-intro-Jr-Devs

In the Terminal:

Download the repo.
```sh
git clone https://github.com/ejonelunas/Git-intro-Jr-Devs.git
cd Git-intro-Jr-Devs
```

Create or edit a file. (Don't forget to save!)
```sh
mkdir contributing-members
touch contributing-members/YOUR_USERNAME.txt
```

"Stage" your changes:
```sh
git add .
```
